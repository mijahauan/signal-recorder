Station Discrimination Methodology

This document explains how GRAPE distinguishes between WWV (Fort Collins, CO) and WWVH (Kekaha, HI) broadcasts on shared frequencies.

Overview

WWV and WWVH broadcast simultaneously on the same frequencies (2.5, 5, 10, 15 MHz). Distinguishing which station is being received requires analyzing multiple signal characteristics. GRAPE uses a weighted voting system with 8 independent methods.

Note: Non-Shared Frequencies

WWV 20 MHz and 25 MHz are WWV-only (WWVH doesn't broadcast). CHU frequencies (3.33, 7.85, 14.67 MHz) are CHU-only. Discrimination is only needed for shared frequencies.

Weighted Voting System

Each method casts a vote for WWV, WWVH, or abstains. Votes are weighted by reliability:

CategoryMethodsWeight
Ground Truth440 Hz Tone, 500/600 Hz Tone, Test Signal10-15 (highest)
Power Ratio1000/1200 Hz Tone Power10 (standard)
CorrelationBCD Time Code, Doppler Stability2-10 (varies)
SupplementaryHarmonic Power Ratio1.5 (confirmatory)

Final Decision: Station with highest weighted vote sum wins. "BALANCED" if margin is below 0.15 threshold.

Ground Truth Methods

Method 6: Test Signal Ground Truth

Schedule: WWV broadcasts at minute :08, WWVH at minute :44

Signal: 100 Hz phase-modulated test signal designed for ionospheric research (HamSCI collaboration)

How it works: Since only one station transmits during these minutes, detection is unambiguous. The signal's presence confirms which station is dominant.

Reliability: Highest - absolute ground truth when detected

Method 3: 440 Hz Station ID Ground Truth

Schedule: WWVH at minute :01, WWV at minute :02

Signal: 440 Hz tone (musical note A above middle C)

How it works: Detection of 440 Hz during minute :01 confirms WWVH; during minute :02 confirms WWV. Only occurs twice per hour.

Reliability: Highest - schedule-based certainty

Method 4: 500/600 Hz Ground Truth Ground Truth

Exclusive WWV minutes: 16, 17, 19

Exclusive WWVH minutes: 43, 44, 45, 46, 47, 48, 49, 50, 51

How it works: During 14 minutes each hour, only one station broadcasts 500/600 Hz tones. Detection during these windows provides unambiguous identification.

Note: WWV broadcasts 500 Hz on even minutes, 600 Hz on odd. WWVH is opposite. The schedule swaps which station has which tone.

Power-Based Methods

Method 1: Timing Tones (1000/1200 Hz) Power Ratio

WWV: 1000 Hz second markers

WWVH: 1200 Hz second markers

How it works: Measures power at both frequencies. The ratio indicates relative signal strength of each station.

ratio_dB = 10 × log₁₀(P_1000 / P_1200)

Positive ratio favors WWV, negative favors WWVH.

Method 1b: Tick SNR (10-sec windows) Power Ratio

Analysis window: 10-second segments

Tick duration: 5 milliseconds

How it works: Integrates tick energy separately at 1000 Hz (WWV) and 1200 Hz (WWVH). Provides fine-grained temporal resolution of which station is stronger.

Correlation-Based Methods

Method 5: BCD Amplitude Correlation Correlation

Subcarrier: 100 Hz amplitude modulation

How it works: Both stations encode UTC time using Binary Coded Decimal (BCD). The patterns are identical but arrive at different times due to propagation delay. Cross-correlation with known BCD templates reveals arrival times.

Detection types:

  • Dual Peak: Both stations received - can measure differential delay
  • Single WWV: Only WWV detected
  • Single WWVH: Only WWVH detected

Method 7: Doppler Stability Correlation

Principle: Ionospheric propagation causes frequency shifts (Doppler). Different paths to WWV vs WWVH produce different Doppler signatures.

Metrics:

  • Doppler Shift (Hz): Instantaneous frequency offset from nominal
  • Doppler Stability (σ): Standard deviation of Doppler over measurement window
  • Coherence Time (Tc): How long the channel remains stable

Advantage: Independent of signal power - useful when both signals have similar strength.

Supplementary Methods

Method 8: Harmonic Power Ratio Supplementary

Harmonics: 500 Hz → 1000 Hz, 600 Hz → 1200 Hz

How it works: The 2nd harmonic of 500 Hz appears at 1000 Hz; 600 Hz appears at 1200 Hz. Since WWV/WWVH swap which station has 500 vs 600 Hz, the harmonic ratio helps distinguish them.

Limitation: Harmonic levels are typically low; supplementary indicator only.

Channel Quality Metrics Supplementary

Source: Test signal at minutes :08 (WWV) and :44 (WWVH)

Metrics:

  • FSS (Frequency Selective Spread): Bandwidth spreading from multipath
  • Delay Spread: Time dispersion of signal arrivals
  • Stability: Consistency of measurements over time
  • Spreading Factor (L): τD × fD - channel physics metric

Purpose: Quality indicators rather than direct discrimination.

BCD Intermodulation Signature Supplementary

Sidebands: 400 Hz = 500-100 Hz, 700 Hz = 600+100 Hz

How it works: The 100 Hz BCD modulation creates sidebands around 500/600 Hz tones. Since WWV and WWVH swap 500/600 Hz by schedule:

  • When WWV has 500 Hz: 400 Hz sideband = WWV, 700 Hz = WWVH
  • When WWV has 600 Hz: 400 Hz sideband = WWVH, 700 Hz = WWV

The ratio (after schedule correction) directly indicates which station is dominant.

Limitation: Sidebands are weak (~10-20 dB above noise); detection rate ~30-40%.

Solar Elevation Overlay

Charts include solar elevation at path midpoints (receiver to WWV, receiver to WWVH). Propagation quality often correlates with solar illumination of the ionosphere.

  • Day: D-layer absorption, shorter skip distances
  • Night: F-layer reflection, longer skip distances
  • Terminator: Rapid changes as ionosphere transitions
